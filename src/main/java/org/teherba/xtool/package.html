xtool is a collection of tools for XML processing with special focus on schema quality and namespace consistency. There is a linear representation of an XSD schema as an XML instance (with possible output to an Excel table), rewriting of namespace prefixes, a namespace URI and prefix crossreference, and an XPath evaluator.
